How it works

All API requests go through this gateway on localhost:8085. The gateway routes to each service via Eureka service discovery.

Swagger UI for each service is proxied through the gateway using prefixed paths:
/iam/swagger-ui/index.html → iam-service's Swagger
/case/swagger-ui/index.html → case-service's Swagger
/workflow/swagger-ui/index.html → workflow-service's Swagger
... and so on for all services.

Note: The Swagger "Try it out" buttons on individual service pages will hit that service's own port directly. To test through the gateway, use Postman with localhost:8085 as the base URL.
